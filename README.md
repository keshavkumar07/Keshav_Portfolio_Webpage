# 📌 About the Project

This is my personal portfolio website developed during my **second year of B.Tech**.

The website presents my profile in a clean and professional way, highlighting my technical skills, projects, education, and career journey. It is fully responsive and designed to provide a smooth experience across desktops, tablets, and mobile devices.

The project focuses on creating an attractive UI while keeping the code simple, organized, and easy to maintain.

---

# 🚀 Getting Started

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/keshav-portfolio.git
```

Or download the ZIP file and extract it.

---

## 2️⃣ Open the Project

Navigate to the project folder.

```bash
cd Keshav_Portfolio_Webpage
```

---

## 3️⃣ Run the Website

Simply open

```text
index.html
```

in your preferred web browser.

For development, you can also use **Live Server** in Visual Studio Code.

# ⚙️ JavaScript Functionality

The website includes several interactive features.

### 📱 Mobile Navigation

- Opens and closes the mobile menu.
- Automatically closes after selecting a navigation link.

---

### ✨ Scroll Reveal Animation

Elements smoothly appear as the user scrolls down the page using the **Intersection Observer API**.

---

### 📊 Animated Skill Bars

When the Skills section becomes visible, the progress bars animate to their respective values.

---

### 🎯 Active Navigation Highlight

The navigation menu automatically highlights the current section while scrolling.

---

### 📌 Sticky Navigation Effect

A subtle shadow appears on the navigation bar while scrolling, improving visibility.
